Work Experience

Between 2017 and 2019 worked as a freelance gemeplay programmer and technical artist.

Since January 2020 I've been working as a Sr Technical Artist at Globant (AAA Contractor); there I've been working since the very start on WB-Avalanche's "Hogwarts Legacy". Where I've worked on Tools, Systems, and Content Optimization:

  • [Editor Tool, Python, C++, Blueprints] Wrote several automated Python Script to set up foliage and collision properties, among others.
  • [Editor Tool, C++, Blueprints] Created smart and lightweight Spline-Based Tool to place collisions on the world (The Main collision tool used throughout the game by Artists and Level Designers).
  • [Editor Tool, C++, Blueprints] Created Automated process to setup and create Mesh Proxies to use in World Composition LOD maps. This tool and its Mesh proxies are how the game handles all the geometry on LOD maps. It is necessary in order to maintain a high quality on Proxies due to sheer size and complexity of level geometry. The tool also uses a Widget Blueprint as its main control interface for setup in the Editor.
  • [Gameplay System, C++, Blueprints] Wrote a runtime system that dynamically populates the world with lightweight impostor trees outside the player near radius. This allows the world to remain fully populated with trees at a lower cost compared to traditional UE4 pipelines.
  • [Editor Tool, Blueprints] Created and supported multiple other smart and configurable spline based tools for mesh actors; such as low walls and fences used throughout the open world.
  • [Teaching, Support] Besides my own tasks, I also regularly helped the teams at Globant, Avalanche, and other Studios within the project with UE4 scripts and general usability questions.
